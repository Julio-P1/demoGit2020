Mi clase de GIT

comando: "git init" para crear el repositorio en el espacio de trabajo
comando: "git status" para verificar cuales archivos estan o no estan es stagin o solo en el respositorio
comando: "git add 'filename' " para agregar el archivo original al stagin
comando: "git rm --cached 'filename'" para hacer unstage al archivo
comando: "git diff -- 'filename'" para ver que parte del codigo del archivo no hemos puesto en staging
comando: "git checkout -- 'filename'" para regresar a la version anterior del codigo antes de haber sido puesto en stagin
comando: "git config --global user.email "juliopaiz@galileo.edu" " para que registre mi correo y se sepa quien es el que hizo el commit
